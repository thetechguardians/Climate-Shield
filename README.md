# Climate-Shield
AI-driven real-time flood and heat risk alert system with crisis mapping and human validation.


# Pre Installation Requirements
1. Sign up and log-in in Twilio website : 
`https://www.twilio.com/en-us`

2. Sign up and log-in in Openweather website : 
`https://home.openweathermap.org/users/sign_in`

2. In Configuration part of `backend/.env`, update the values



# Installation
`pip install requests pandas numpy prophet twilio`

then click on run python file
